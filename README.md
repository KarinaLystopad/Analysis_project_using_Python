# Проєкт: Аналіз великих датасетів з Pandas

## Опис
У цьому репозиторії зберігаються результати виконання проєкту з аналізу даних за допомогою бібліотеки **Pandas** та методів описової статистики.

## Мій внесок
У межах проєкту я:
- Працювала з великими наборами даних, використовуючи Pandas.
- Застосовувала методи описової статистики та агрегації даних.
- Аналізувала міри центральної тенденції та відсоткові розподіли.
- Виконувала групування та фільтрацію даних за різними критеріями.
- Обробляла пропущені значення та множини.
- Візуалізувала результати аналізу у зручному табличному форматі.

## Структура файлів
- `KLystopad_PythonProject.ipynb` — Jupyter Notebook із прикладами роботи з датасетом, застосуванням статистичних методів та візуалізацією результатів.
